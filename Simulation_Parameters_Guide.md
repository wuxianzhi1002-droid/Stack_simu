# Lumerical 仿真设置参数详细步骤指南

本指南详细介绍了项目中自动生成的 `.fsp` 文件的设置原则，帮助你在 Lumerical GUI 界面中理解仿真配置。

## 1. 宽带光源设置 (Broadband Source)
在生成的 `.fsp` 文件中，光源被配置为 **Plane Wave** (平面波)：
- **光源类型**: `Plane wave`
- **入射方向**: `Backward` (沿 -z 方向向下入射)，模拟光束从空气射向膜层。
- **波长范围**: 400nm - 800nm。
- **设置原则**: 
    - 调焦调平通常使用宽带光以减弱单色光的强干涉波动，获得更平滑的信号。
    - 平面波模拟了 1D 无限大平面的入射情况，完全考虑了光在多层膜之间的**相干叠加**和多重反射。

## 2. 几何结构与网格 (Geometry & Mesh)
- **堆叠顺序**: 按照 `stack_builder.py` 定义，从底部（Substrate）到顶部（HSQ）依次排列。
- **维度**: 使用 **2D Z-X** 仿真。对于 1D 平面膜层，2D 仿真与 3D 结果一致，但计算速度快 100 倍以上。
- **网格原则**: 
    - 软件自动根据材料折射率划分网格（每个波长至少 10 个网格点）。
    - 对于 5nm 的超薄层，建议在界面手动添加 `Mesh Override`（虽然脚本已尽量优化，但手动微调 Z 轴网格至 1nm 可获得最高精度）。

## 3. 边界条件 (Boundary Conditions)
- **Z轴 (上下)**: 使用 **PML (Standard)**。吸收掉离开仿真区域的光，模拟无限空间。
- **X轴 (左右)**: 使用 **Periodic**。模拟膜层在横向无限延伸。

## 4. 监视器 (Monitors)
- **R_monitor**: 放置在光源上方。通过计算通过该平面的净功率流来获取**反射光谱**。
- **T_monitor**: 放置在基底（Substrate）内部。获取**透射光谱**。
- **数据处理**: 监视器会自动执行傅里叶变换，将时域场转换为频域响应，从而给出 400-800nm 的连续光谱。

## 5. 相干性考虑 (Coherence)
- **TMM (STACK)**: 本项目核心计算使用的 `stackrt` 函数基于传输矩阵法，它是严格的波动光学解，**完全考虑了各层界面间的相位差和相干干涉**。
- **FDTD**: 生成的 `.fsp` 文件如果运行仿真，其结果应与 STACK 计算出的反射率完全吻合，进一步验证了相干性计算的准确性。
